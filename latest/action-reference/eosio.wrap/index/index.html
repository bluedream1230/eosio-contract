wrap

Type: class

eosio.wrap contract simplifies Block Producer superuser actions by making them more readable and easier to audit.

system. Currently, 15/21 block producers can already change an account's keys or modify an account's contract at the request of ECAF or an account's owner. However, the current method is opaque and leaves undesirable side effects on specific system accounts. eosio.wrap allows for a cleaner method of implementing these important governance actions.
